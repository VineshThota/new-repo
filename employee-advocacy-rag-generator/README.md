# 🚀 AI-Powered Employee Advocacy Content Generator

## Overview

The AI-Powered Employee Advocacy Content Generator is a cutting-edge web application that combines the trending LinkedIn topic of **Employee Advocacy** with **RAG (Retrieval-Augmented Generation)** technology to create personalized, brand-aligned LinkedIn content for employees.

## 🎯 Problem Solved

Companies struggle to create personalized, context-aware content for employee advocacy programs that align with their brand voice and industry expertise. This application addresses the challenge by:

- **Automating Content Creation**: Eliminates the time-consuming process of manually crafting LinkedIn posts
- **Ensuring Brand Consistency**: Uses RAG technology to maintain brand voice across all employee content
- **Personalizing at Scale**: Creates unique content tailored to individual employees and their roles
- **Optimizing Engagement**: Incorporates LinkedIn best practices and trending formats

## ✨ Key Features

### 🧠 RAG-Powered Content Generation
- **Retrieval-Augmented Generation**: Analyzes company brand guidelines and industry knowledge
- **Context-Aware Content**: Generates posts that align with company values and messaging
- **Industry-Specific Terminology**: Incorporates relevant industry language and trends

- **Intuitive Design**: Clean, professional interface optimized for ease of use
- **Real-Time Preview**: Instant preview of generated LinkedIn posts

- **Mobile Responsive**: Works seamlessly across all devices

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/VineshThota/new-repo.git
   ```

2. Navigate to the project directory:
   ```bash
   cd new-repo/employee-advocacy-rag-generator
   ```

3. Open `index.html` in your web browser or serve it using a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

### Usage Guide

1. **Company Configuration**:
   - Enter your company name
   - Select your industry from the dropdown
   - Describe your brand voice and guidelines

2. **Employee Details**:
   - Input employee name and role
   - This personalizes the generated content

3. **Content Specification**:
   - Choose content type (company news, insights, etc.)
   - Provide specific topic or news to feature

4. **Generate Content**:
   - Click "Generate AI Content"
   - Wait for RAG processing (3-second simulation)
   - Review generated LinkedIn post

5. **Content Actions**:
   - Copy content to clipboard
   - Regenerate for alternative versions
   - Monitor engagement predictions
